# Security Tools

## 1. Overview

Security Tools is a collection of cybersecurity projects built from scratch to develop practical skills in networking, programming, operating systems, offensive security, and defensive security.

The objective of this repository is to learn by building real tools rather than relying solely on tutorials or existing libraries. Each project focuses on understanding how security tools work internally and implementing the underlying concepts manually whenever practical.

---

## 2. Goals

* Strengthen programming skills through real-world projects.
* Develop a deeper understanding of networking and operating systems.
* Learn offensive and defensive security techniques.
* Practice software engineering principles, including modular design and testing.
* Build a public portfolio demonstrating practical cybersecurity knowledge.

---
